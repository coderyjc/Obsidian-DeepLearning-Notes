## 如何使用？

本仓库包含了70MB的Obsidian插件，因此占据空间非常大。

因此，可以使用浏览器插件`gitzip`选择性下载。[chrome应用商店链接](https://chromewebstore.google.com/detail/gitzip-for-github/ffabmkklhbepgcgfonabamgnfafbdlkn)
